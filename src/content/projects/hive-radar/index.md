Hive Radar is positioned as an advanced competitor intelligence tool for local businesses and agencies evaluating market opportunity.

## Overview

The interface is built around a focused analysis workflow: define target business context, keywords, and locations, then run a market-opportunity scan.

## Core Capabilities

- Competitor intelligence and revenue leakage scanning
- Industry quick-add presets for common local service categories
- Optional GMB-verified target business selection
- Auto-fill of nearby cities within a 50-mile radius when GMB data is selected
- Optional website input for technical audit context

## Supported Market Inputs

- Target keywords
- Target locations
- Average customer value
- Optional website URL for technical audit framing

## Intended Users

- Local SEO teams qualifying expansion opportunities
- Agencies benchmarking service niches across cities
- Service businesses validating keyword and location strategy before spend
